

Terminology:
session: Recording data of the same neurons. Consisting of several trials.
trial: A continuous duration of recording. Typically one trial in task, or one continuous eyes-open/eyes-closed duration.
session_name: Text mark of a session. Contains control type (Mus/Sal) and recorded area. 
session_idx: Recording dates (MMDDYYYY) is converted to session index. Sessions under the same name and idx contains the same group of neurons.
state: Specific period in a session: Task/RestOpen/RestClose, Pre/Post(injection).
